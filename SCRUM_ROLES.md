# 👤 Scrum Team Roles — APMS (Ashar Grosir Perfume Management System)

> **Project Type:** Solo Developer + AI Agent Assistant  
> **Developer:** Ashar (Solo)  
> **AI Assistant:** Antigravity AI Agent (supporting documentation, code review, and implementation guidance)  
> **Project Duration:** April 2026  
> **Tech Stack:** Laravel 11, MySQL, TailwindCSS, Blade Templates  

---

## 🎯 Role Assignment

Since this is a **solo developer project**, all Scrum roles are assumed by one person (Ashar), with AI acting as a supportive collaborator for technical decisions.

---

### 🏆 Product Owner — Ashar
**Responsibilities:**
- Defines the vision and goals of the APMS application
- Owns and prioritizes the Product Backlog
- Decides which features are "Done" and accepted
- Represents the end user (toko parfum owner/staff needs)
- Makes business decisions (e.g., which reports are needed, how wholesale works)

**Why Ashar as PO:**  
As the owner of the perfume store, Ashar best understands the business requirements — what the cashier needs, how inventory should be tracked, and what reports matter for daily operations.

---

### 🔧 Scrum Master — Ashar (+ AI Agent Support)
**Responsibilities:**
- Facilitates daily stand-up logs (written in repo)
- Removes blockers and impediments (e.g., debugging errors, resolving dependency issues)
- Ensures Scrum ceremonies are followed (Sprint Planning, Review, Retrospective)
- Tracks sprint velocity and progress

**AI Agent Role as Scrum Support:**
- Assists in writing sprint documentation
- Helps identify blockers and propose solutions
- Supports code review to maintain quality standards

---

### 💻 Developer — Ashar (+ AI Agent Pair Programmer)
**Responsibilities:**
- Implements all features defined in the Product Backlog
- Writes and maintains database migrations
- Builds Laravel controllers, models, views, and routes
- Performs self-testing and QA before marking tasks as Done
- Commits daily to GitHub repository

**AI Agent Role as Pair Programmer:**
- Assists with code generation and bug fixing
- Reviews code for logic errors and best practices
- Proposes architecture improvements

---

## 🤖 AI Agent Contribution Policy
The AI Agent (Antigravity) acts as an **assistive tool**, not an autonomous developer. All final decisions, commits, and approvals are made by Ashar. The AI provides:
- Code suggestions and templates
- Bug diagnosis and fix recommendations
- Documentation drafting
- UI/UX design guidance

> *"Solo doesn't mean alone — AI-assisted development is a modern, valid approach to agile software delivery."*
